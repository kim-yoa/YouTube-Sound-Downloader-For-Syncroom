## 소개
[yt-dlp](https://github.com/yt-dlp/yt-dlp)와 [FFmpeg](https://ffmpeg.org/)를 사용하여 유튜브 영상을 [Syncroom](https://syncroom.yamaha.com/global/)의 미디어 플레이어로 사운드를 재생시킬 수 있는 파일(wav, mp3)로 다운로드하는 소프트웨어입니다.
